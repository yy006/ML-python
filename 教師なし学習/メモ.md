## 因子分析

------
データの相関行列に見出すことができる塊(相関の関係)から、背後の構造を分析する

例）2つの因子が5つのデータにかかわっている[動画](https://www.youtube.com/watch?v=ZJfytzsqI1o&list=PLhDAH9aTfnxJGiLwq2CRS87J51ZPU5IEM)のような設定とする。

A = f_0             + ε_A
B = f_0             + ε_B
C = 1/2f_0 + 1/2f_1 + ε_C
D =          f_1    + ε_D
E =          f_1    + ε_E
(因子f_0の影響) + (因子f_1の影響) + (二つの因子以外からの影響)とあらわせる、という分析
